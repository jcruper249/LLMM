Introduccion

Web4U es una pagina web para aprender sobre desarrollo web.

En esta pagina podras encontrar informacion sobre HTML, CSS y SVG.

  • HTML: Es el lenguaje de marcado que se utiliza para estructurar el contenido de una pagina web.
  • CSS: Es el lenguaje de estilos que se utiliza para dar formato y diseño a una pagina web.
  • SVG: Es un formato de imagen vectorial que se utiliza para crear graficos y dibujos en una pagina web.
