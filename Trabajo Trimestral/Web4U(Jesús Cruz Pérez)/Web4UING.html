Introduction

Web4U is a website to learn about web development.

On this page you can find information about HTML, CSS, and SVG.

  • HTML: It is the markup language used to structure the content of a web page.
  • CSS: It is the style language used to give format and design to a web page.
  • SVG: It is a vector image format used to create graphics and drawings on a web page.
