CSS

CSS Syntax

A CSS rule consists of a selector and a declaration block:

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

CSS code example:

p {

color: red;

text-align: center;

}

CSS Selectors

CSS selectors are used to "select" the HTML elements you want to style.

We can divide CSS selectors into five categories:

  • Simple selectors (select elements by name, id, class)
  • Combinator selectors: (select elements based on a specific relationship between them)
  • Pseudo-class selectors: (select elements based on a certain state)
  • Pseudo-element selectors: (select and style a part of an element)
  • Attribute selectors: (select elements based on an attribute or attribute value)

Let's explain 2 of the most basic ones:

The CSS element selector

The element selector selects HTML elements by element name.

CSS code example:

Here all the elements of the page will be centered with red text color:

p {

color: red;

text-align: center;

}

The CSS id selector

The id uses the identification attribute of an HTML element to select a specific element.

An element's id is unique within a page, so the id selector is used to select a unique element!

To select an element with a specific id, write a hash character (#), followed by the element's id.

CSS code example:

The following CSS rule will apply to the HTML element with id="introduccion":

#introduccion {

color: blue;

text-align: center;

}

How to Add CSS

There are three ways to add CSS to an HTML document:

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

External CSS is defined in a separate CSS file with a .css extension.

The external CSS file is linked to an HTML document using the <link> element.

CSS code example:

External styles are defined inside the <link> element, within the <head> section of an HTML page:

<!DOCTYPE html>

<html>

<head>

<link rel="stylesheet" href="mystyle.css">

</head>

<body>

<h1>This is a heading</h1>

<p>This is a paragraph.</p>

</body>

</html>

An external style sheet can be written in any text editor and must be saved with a .css extension.

The external .css file must not contain HTML tags.

This is how the file "mystyle.css" looks:

"mystyle.css"

h1 {

color: blue;

}

p {

color: red;

}

Internal CSS

An internal style sheet can be used if a single HTML page has a unique style.

The internal style is defined inside the <style> element, within the <head> section of an HTML page.

CSS code example:

Internal styles are defined inside the <style> element, within the <head> section of an HTML page:

<!DOCTYPE html>

<html>

<head>

<style>

h1 {

color: blue;

}

p {

color: red;

}

</style>

</head>

<body>

<h1>This is a heading</h1>

<p>This is a paragraph.</p>

</body>

</html>

Inline CSS

Inline CSS is used to apply a unique style to a single HTML element.

To add inline styles, use the style attribute inside the HTML element.

CSS code example:

The following example applies inline styles to the <h1> element and the <p> element:

<!DOCTYPE html>

<html>

<body>

<h1 style="color:blue;">This is a heading</h1>

<p style="color:red;">This is a paragraph.</p>

</body>

</html>

CSS Comments

Comments are used to explain CSS code and can help you edit the source code at a later date.

Comments are also used to temporarily disable parts of the CSS code inside a style sheet.

Browsers ignore comments!

A CSS comment is placed within the HTML <style> element, and starts with /* and ends with */:

CSS code example:

/* This is a single-line comment */

p {

color: red;

}

CSS Box Model

In CSS, the term "box model" is used when talking about layout and web design.

The CSS box model is essentially a box that wraps around each HTML element.

Each box consists of four parts: content, padding, borders, and margins.

The following image illustrates the CSS box model:

Explanation of the different parts (from innermost to outermost):

  • Content: The content of the box, where text and images appear.
  • Padding: The space between the box content and the box border. Padding is transparent.
  • Border: The border that surrounds the padding (if defined). The border is visible.
  • Margin: The space between the box border and the boxes of other elements. Margins are transparent.

The box model allows us to add a border around elements and define the space between elements.

CSS code example:

div {

width: 300px;

border: 25px solid navy;

padding: 25px;

margin: 25px;

}
