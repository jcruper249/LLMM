HTML

Encabezados HTML

Los encabezados HTML se definen con <h1> a <h6> etiquetas

Ejemplo de código HTML:

<h1>Este es el título 1</h1>

<h2>Este es el título 2</h2>

<h3>Este es el título 3</h3>

<h4>Este es el título 4</h4>

<h5>Este es el título 5</h5>

<h6>Este es el título 6</h6>

Párrafos HTML

Los párrafos HTML se definen con la etiqueta <p>

Ejemplo de código HTML:

<p>Este es un párrafo de ejemplo.</p>

Enlaces HTML

Los enlaces HTML se definen con la etiqueta <a>

Ejemplo de código HTML:

<a href="https://japanpaw.com/">Este es un enlace de ejemplo.</a>

Enlaces HTML - Sintaxis

El HTML <a> la etiqueta define un hipervínculo. Tiene la siguiente sintaxis:

Ejemplo de código HTML:

<a href="URL del enlace">Texto del enlace</a>

El atributo más importante del <a> el elemento es el atributo href que indica el destino del enlace.

El enlace texto es la parte que será visible para el lector.

Al hacer clic en el texto del enlace, se enviará al lector a la dirección URL especificada.

Ejemplo de código HTML:

<a href="https://japanpaw.com/">Visita Japan Paw</a>

Imágenes HTML

Las imágenes HTML se definen con la etiqueta <img>

El archivo fuente (src), text alternativ (alt), ancho, altura se proporcionan como atributos:

Ejemplo de código HTML:

<img src="imagen.jpg" alt="Descripción de la imagen">

HTML Styles

Los estilos HTML se definen con el atributo style

Ejemplo de código HTML:

<h1 style="color:blue;">Este es un título azul</h1>
