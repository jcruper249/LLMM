SVG Circle

SVG circles are defined with the <circle> tag.

Attribute Description
cx Defines the x-position of the circle's center.
cy Defines the y-position of the circle's center.
r Defines the radius of the circle.
SVG code example:

<svg width="100" height="100">

<circle cx="50" cy="50" r="40" fill="red" />

</svg>

SVG Ellipse - <ellipse>

SVG ellipses are defined with the <ellipse> tag.

An ellipse is closely related to a circle. The difference is that an ellipse has an x-radius and a y-radius that differ, while a circle has equal radii.

The <ellipse> element has four basic attributes used to position and size the ellipse:

Attribute Description
cx The x-axis center of the ellipse. Default is 0.
cy The y-axis center of the ellipse. Default is 0.
rx Required. The x-radius of the ellipse.
ry Required. The y-radius of the ellipse.
SVG code example:

<svg width="200" height="100">

<ellipse cx="100" cy="50" rx="80" ry="40" fill="orange" />

</svg>

SVG Line - <line>

SVG lines are defined with the <line> tag.

The <line> element creates a line between the starting position (x1,y1) and the ending position (x2,y2).

The <line> element has four basic attributes for positioning and determining the length of the line:

Attribute Description
x1 The x-axis start of the line.
y1 The y-axis start of the line.
x2 The x-axis end of the line.
y2 The y-axis end of the line.
SVG code example:

<svg width="200" height="100">

<line x1="0" y1="0" x2="200" y2="100" stroke="red" />

</svg>

SVG Polygon - <polygon>

The <polygon> element is used to create a graphic containing at least three sides.

Polygons are made of straight lines, and the shape is “closed” (the last point automatically connects to the first).

The <polygon> element has one basic attribute that defines the polygon's points:

Attribute Description
points Required. The list of the polygon's points. Each point must contain an x and y coordinate.
SVG code example:

<svg width="200" height="200">

<polygon points="100,10 150,190 50,190" stroke="black" stroke-width="1" fill="lime" />

</svg>

SVG Polyline - <polyline>

The <polyline> element is used to create a series of connected lines.

Unlike the <polygon> element, the <polyline> element does not close the shape automatically (it does not connect the last point to the first).

Attribute Description
points Required. The list of the polyline’s points. Each point must contain an x and y coordinate.
SVG code example:

<svg width="200" height="200">

<polyline points="0,0 50,150 100,75 150,50 200,140 250,140" stroke="blue" stroke-width="2" fill="none" />

</svg>

SVG Rectangle - <rect>

SVG rectangles are defined with the <rect> tag.

The <rect> element has four basic attributes for positioning and sizing the rectangle:

Attribute Description
x The x-position of the rectangle’s top-left corner.
y The y-position of the rectangle’s top-left corner.
rx The x-axis radius for rounded corners.
ry The y-axis radius for rounded corners.
width Required. The width of the rectangle.
height Required. The height of the rectangle.
SVG code example:

<svg width="200" height="100">

<rect x="10" y="10" width="150" height="80" stroke="black" fill="blue" />

</svg>
