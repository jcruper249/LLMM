CSS

Sintaxis CSS

Una regla CSS consta de un selector y un bloque de declaración:

El selector apunta al elemento HTML que deseas estilizar.

El bloque de declaración contiene una o más declaraciones separadas por punto y coma.

Cada declaración incluye un nombre de propiedad CSS y un valor, separados por dos puntos.

Ejemplo de código CSS:

p {

color: red;

text-align: center;

}

Selectores CSS

Los selectores CSS se utilizan para "seleccionar" los elementos HTML que deseas estilizar.

Podemos dividir los selectores CSS en cinco categorías:

  • Selectores simples (seleccione elementos según nombre, identificación, clase)
  • Selectores combinados: (seleccionar elementos basados en una relación específica entre ellos)
  • Selectores de pseudo-clase: (seleccione elementos según un estado determinado)
  • Selectores de pseudo-elemento: (seleccionar y estilizar una parte de un elemento)
  • Selectores de atributo: (seleccione elementos según un atributo o valor de atributo)

Vamos a explicar 2 de los mas basicos:

El selector de elementos CSS

El elemento el selector selecciona elementos HTML según el nombre del elemento.

Ejemplo de código CSS:

Aquí estarán todos los elementos de la página alineado con el centro, con un color de texto rojo:

p {

color: red;

text-align: center;

}

El selector de id CSS

El id utiliza el atributo de identificación de un elemento HTML para seleccionar un elemento específico.

La id de un elemento es única dentro de una página, por lo que el selector de identificación lo es acostumbrado a ¡selecciona un elemento único!

Para seleccionar un elemento con una identificación específica, escriba un carácter hash (#), seguido de la identificación del elemento.

Ejemplo de código CSS:

La siguiente regla CSS se aplicará al elemento HTML con id="introduccion":

#introduccion {

color: blue;

text-align: center;

}

Cómo agregar CSS

Hay tres formas de agregar CSS a un documento HTML:

  • Css externo
  • Css interno
  • Css en línea

Css externo

El CSS externo se define en un archivo CSS separado con una extensión .css.

El archivo CSS externo se vincula a un documento HTML mediante el elemento <link>.

Ejemplo de código CSS:

Los estilos externos se definen dentro del elemento <link>, dentro de la sección <head> de una página HTML:

<!DOCTYPE html>

<html>

<head>

<link rel="stylesheet" href="mystyle.css">

</head>

<body>

<h1>Este es un título</h1>

<p>Este es un párrafo.</p>

</body>

</html>

Una hoja de estilo externa se puede escribir en cualquier editor de texto y debe guardarse con una extensión .css.

El archivo .css externo no debe contener etiquetas HTML.

Así se ve el archivo "mystyle.css":

"mystyle.css"

h1 {

color: blue;

}

p {

color: red;

}

Css interno

Se puede utilizar una hoja de estilo interna si una sola página HTML tiene un estilo único.

El estilo interno se define dentro del elemento <style>, dentro de la sección <head> de una página HTML.

Ejemplo de código CSS:

Los estilos internos se definen dentro del elemento <style>, dentro de la sección <head> de una página HTML:

<!DOCTYPE html>

<html>

<head>

<style>

h1 {

color: blue;

}

p {

color: red;

}

</style>

</head>

<body>

<h1>Este es un título</h1>

<p>Este es un párrafo.</p>

</body>

</html>

Css en línea

El CSS en línea se utiliza para aplicar un estilo único a un solo elemento HTML.

Para agregar estilos en línea, utilice el atributo style dentro del elemento HTML.

Ejemplo de código CSS:

El siguiente ejemplo aplica estilos en línea al elemento <h1> y al elemento <p>:

<!DOCTYPE html>

<html>

<body>

<h1 style="color:blue;">Este es un título</h1>

<p style="color:red;">Este es un párrafo.</p>

</body>

</html>

Comentarios del CSS

Los comentarios se utilizan para explicar el código CSS y pueden ayudarle a editarlo código fuente en una fecha posterior.

Los comentarios también se utilizan para deshabilitar temporalmente secciones del código CSS dentro de a hoja de estilo.

¡los navegadores ignoran los comentarios!

Un comentario CSS se coloca dentro del HTML <style> elemento, y comienza con /* y termina con */:

Ejemplo de código CSS:

/* Este es un comentario de una sola línea */

p {

color: rojo;

}

CSS Modelo de caja

En CSS, el término "modelo de caja" se utiliza cuando se habla de diseño y maquetación web.

El modelo de caja CSS es esencialmente una caja que envuelve cada elemento HTML.

Cada caja consta de cuatro partes: contenido, relleno, bordes y márgenes.

La siguiente imagen ilustra el modelo de caja CSS:

Explicación de las diferentes partes (de la parte más interior a la parte más exterior):

  • Contenido: El contenido de la caja, donde el texto y las imágenes aparecen.
  • Relleno: El espacio entre el contenido de la caja y el borde de la caja. El relleno es transparente.
  • Borde: El borde que rodea el relleno (si se ha definido). El borde es visible.
  • Márgenes: El espacio entre el borde de la caja y las cajas de otros elementos. Los márgenes son transparentes.

El modelo de caja nos permite agregar un borde alrededor de los elementos y definir el espacio entre elementos.

Ejemplo de código CSS:

div {

width: 300px;

border: 25px solid navy;

padding: 25px;

margin: 25px;

}
