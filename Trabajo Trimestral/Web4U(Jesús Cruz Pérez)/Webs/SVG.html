Círculo SVG

Los círculos SVG se definen con la etiqueta <circle>

Atributo Descripción
cx Define la posición x del centro del círculo.
cy Define la posición y del centro del círculo.
r Define el radio del círculo.
Ejemplo de código SVG:

<svg width="100" height="100">

<circle cx="50" cy="50" r="40" fill="red" />

</svg>

SVG Ellipse - <ellipse>

Las elipses SVG se definen con la etiqueta <ellipse>

Una elipse está estrechamente relacionada con un círculo. La diferencia es que una elipse tiene un radio x e y que difieren entre sí, mientras que un círculo tiene x igual y radio y.

El <ellipse> el elemento tiene cuatro atributos básicos para posicionar y establecer tamaño de la elipse:

Atributo Descripción
cx El centro del eje x de la elipse. El valor predeterminado es 0
cy El centro del eje y de la elipse. El valor predeterminado es 0
rx Requerido. El radio x de la elipse
ry Requerido. El radio y de la elipse
Ejemplo de código SVG:

<svg width="200" height="100">

<ellipse cx="100" cy="50" rx="80" ry="40" fill="orange" />

</svg>

SVG Linea - <line>

Las líneas SVG se definen con la etiqueta <line>

El <line> el elemento crea una línea entre la posición inicial (x1,y1) y la posición final (x2,y2).

El <line> el elemento tiene cuatro atributos básicos para posicionar y establecer longitud de la línea:

Atributo Descripción
x1 El inicio de la línea en el eje x
y1 El inicio de la línea en el eje y
x2 El final de la línea en el eje x
y2 El final de la línea en el eje y
Ejemplo de código SVG:

<svg width="200" height="100">

<line x1="0" y1="0" x2="200" y2="100" stroke="red" />

</svg>

SVG Poligono - <polygon>

El <polygon> el elemento se utiliza para crear un gráfico que contenga al menos tres lados.

Los polígonos están hechos de líneas rectas y la forma es "cerrada" (conecta automáticamente el último punto con el primero).

El <polygon> el elemento tiene un atributo básico eso define los puntos del polígono:

Atributo Descripción
Puntos Requerido. La lista de puntos del polígono. Cada punto debe contener un coordenada x y coordenada y
Ejemplo de código SVG:

<svg width="200" height="200">

<polygon points="100,10 150,190 50,190" stroke="black" stroke-width="1" fill="lime" />

</svg>

SVG Polilínea - <polyline>

El <polyline> el elemento se utiliza para crear una serie de líneas conectadas.

A diferencia del <polygon> el elemento, el <polyline> el elemento no cierra automáticamente la forma (no conecta el último punto con el primero).

Atributo Descripción
Puntos Requerido. La lista de puntos de la polilínea. Cada punto debe contener un coordenada x y coordenada y
Ejemplo de código SVG:

<svg width="200" height="200">

<polyline points="0,0 50,150 100,75 150,50 200,140 250,140" stroke="blue" stroke-width="2" fill="none" />

</svg>

SVG Rectángulo - <rect>

Los rectángulos SVG se definen con la etiqueta <rect>

El <rect> el elemento tiene cuatro atributos básicos para posicionar y establecer tamaño del rectángulo:

Atributo Descripción
x La posición x para la esquina superior izquierda del rectángulo
y La posición y para la esquina superior izquierda del rectángulo
rx El radio de las esquinas redondeadas en el eje x
ry El radio de las esquinas redondeadas en el eje y
width Requerido. El ancho del rectángulo
height Requerido. La altura del rectángulo
Ejemplo de código SVG:

<svg width="200" height="100">

<rect x="10" y="10" width="150" height="80" stroke="black" fill="blue" />

</svg>
