HTML

HTML Headings

HTML headings are defined with the <h1> to <h6> tags.

HTML code example:

<h1>This is heading 1</h1>

<h2>This is heading 2</h2>

<h3>This is heading 3</h3>

<h4>This is heading 4</h4>

<h5>This is heading 5</h5>

<h6>This is heading 6</h6>

HTML Paragraphs

HTML paragraphs are defined with the <p> tag.

HTML code example:

<p>This is an example paragraph.</p>

HTML Links

HTML links are defined with the <a> tag.

HTML code example:

<a href="https://japanpaw.com/">This is an example link.</a>

HTML Links - Syntax

The HTML <a> tag defines a hyperlink. It has the following syntax:

HTML code example:

<a href="Link URL">Link text</a>

The most important attribute of the <a> element is the href attribute, which indicates the link destination.

The link text is the part that will be visible to the reader.

When clicking the link text, the reader will be sent to the specified URL.

HTML code example:

<a href="https://japanpaw.com/">Visit Japan Paw</a>

HTML Images

HTML images are defined with the <img> tag.

The source file (src), alternative text (alt), width, and height are provided as attributes:

HTML code example:

<img src="image.jpg" alt="Image description">

HTML Styles

HTML styles are defined with the style attribute.

HTML code example:

<h1 style="color:blue;">This is a blue title</h1>
